This project demonstrate the use of C++ classes, constructor, destructors, and file I/O that was taught in class.

          when the object reaches the end of its lifecycle, it is destroyed by the destructor "~Product."

The project shows the material learned during week 5, also it includes the separation of the material:
header/cpp documents have been included.
