What does this program do?
This program implements a simple inventory system, this has been implemented using C++. 

This program let you save items to a file by using the saveToFile() method from the class "Item."

It lets you to retrieve any information by using the loadFromFile() method that is part of the class "Item."

Files Included:
Items.h - Header of the program.
Items.cpp - Where the main implementation of the methods are located.
main.cpp - where the main() function is located.
