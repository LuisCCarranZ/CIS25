function_loop.h: this is the header of the program.
Function_and_Loops.cpp: this is the main program that needs to be executed.

This program implements a simple inventory menu system. The user can add up to 10 items. 
The total limit of slots is displayed when adding more items. 
Once that limit has been reached, the program only let the user to display the items that were added. 
The user can exit the program by selecting option three and then confirming it knowing that the items
stored are going to be destroyed.
